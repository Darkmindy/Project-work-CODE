- Login Page con strong authentication per garantire la sicurezza degli utenti

- Checkout
- schermata di pagamento
- Admin Page
- Post e put dei prodotti
- Delete prodotti
- Funzione di ricerca
- Salvataggio ordini

- JSON Server
- TypeScript
- Bootstrap
